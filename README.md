* Contas Pagar 
* Contas Receber
* Estoque

## Produtos 
Dados dos produtos como:
  * Nome 
  * Preços
  * Quantidade
  * Tipo 
  

Ao adicionar o sistema vai pedir a quantidade e o preço dessa quantidade

**Ex:** 1 caixa de ovos / 10 reais -> 12 ovos / 10 reais -> 1.2 reais / ovo
